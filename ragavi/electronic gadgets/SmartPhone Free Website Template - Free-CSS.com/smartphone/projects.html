Features

The Smartphone Bold smartphone embodies elegant design - without sacrificing the features or functionality you expect from a premium smartphone.

Learn about the Smartphone Bold smartphone features.

Specifications

  • Wireless Email, Internet, SMS / MMS
  • Keyboard backlighting
  • Integrated earpiece/ microphone
  • TFT LCD resln. 480 x 320 pixel color display
  • GSM Standby Time: 9 days
  • Expandable memory - support microSD card
  • Wi-Fi: 802.11b/g USB cable, Wall Charger
